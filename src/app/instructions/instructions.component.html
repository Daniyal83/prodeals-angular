Instructions

Aliexpress

  • If you click the button you'll be redirected to the seller's page.
  • Sign in if you're registered or register an account.
  • Select bundle, color, shipping country and quantity of the product. Then click the button.
  • On the next page set the shipping information. If you have any coupons apply them.
  • If there is no % icon just pay for the order.
  • If you see the % icon then you can get a discount or a free gift.
  • To get a disount or a gift you should leave message for the seller and place order without payment.
  • As soon as you see that final price is reduced you can pay for the order.
